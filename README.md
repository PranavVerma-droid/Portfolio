## Docker Deploy 🐋
To run this code via **docker**, pull the repository via:

```bash
git clone https://github.com/PranaVerma-droid/Portfolio.git
```

and then run this to deploy via docker:
```bash
docker compose up -d
```

## Manual Deploy 💪
To deploy without Docker, run:
```bash
npm install
npm start
```
